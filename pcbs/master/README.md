# Master board
![alt text](image.png)
## Features
* STM32F405 MCU
* 4x RS485 ports
* 5-24v main power input
* 12A TPS513885-powered 5v regulated power rail
* 3x Neopixel LEDs
* SWD & USB programming support

## Assembly
Solder on all SMT parts first!
- For the buck converter chip and STM32, I recommend first melting solder paste onto the pads, then cleaning it up with a soldering iron before using a hot air station to solder the chip on
- For small passive/discrete components, use the classic method by tacking on paste first and soldering with hot air
- Then solder on through-hole components (connectors, capacitors) with a soldering iron
